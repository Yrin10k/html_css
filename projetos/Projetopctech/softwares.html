Winrar

O WinRAR é um dos mais famosos programas para compactar e descompactar arquivos.
O aplicativo tem suporte aos principais formatos de compressão, como RAR, ZIP e ISO, tem interface simples e oferece a possibilidade de recuperar arquivos danificados.

GIMP

O GIMP é uma ferramenta multi-plataforma de manipulação de fotos. GIMP é um acrônimo para GNU Image Manipulation Program (Programa de Manipulação de Imagens GNU). O GIMP é adequado para várias tarefas de manipulação de imagens, incluindo retocar fotos, composição e construção de imagens. O GIMP tem muitos recursos.

Utorrent

O uTorrent é um cliente BitTorrent, este um sistema de compartilhamento de arquivos P2P (peer-to-peer, ou ponto-a-ponto), onde todos os usuários da rede atuam como servidores e clientes, distribuindo fragmentos dos arquivos, geralmente mídias.
